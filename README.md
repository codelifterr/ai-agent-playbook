# AI Agent Playbook

Python utilities and examples for AI agent patterns, workflow design, and risk checks.

## Features

- Define reusable agent patterns
- Classify task risk before automation
- Generate lightweight workflow plans
- Keep human approval for high-risk actions

## Run locally

```bash
python3 -m ai_agent_playbook.cli "research a company" --risk medium
```

## Why this project exists

AI agents are most useful when they are structured, verifiable, and safe. This project turns common agent ideas into small readable Python code.
